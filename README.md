# TECH-STACK
  - **Database** - MySQL
  - **Backend** - Python , Flask
  - **Frontend** - Streamlit
  - **DB Tool** - My SQL Workbench

# FEATURES
```text
  - User registration and login with hashed passwords
  - Product and category management 
  - Order placement using ACID transactions (stored procedure)
  - Auto stock reduction on order (trigger)
  - Auto order cancellation on payment failure (trigger)
  - Product ratings summary (view)
  - Order history per user (view)
  - Indexed queries for performance
```

  # DATABASE LAYER
    8 normalized tables
    2 triggers
    1 stored procedure
    2 views
    5 indexes
